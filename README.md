# Ais Liquid Clarity - Temizlik Çözümleri

Modern ve şık bir temizlik ürünleri şirketi web sitesi.

## Kurulum

```bash
npm install
```

## Geliştirme

Geliştirme sunucusunu başlatmak için:

```bash
npm run dev
```

Tarayıcıda `http://localhost:3000` adresinde görüntüleyebilirsiniz.

## Build

Production build almak için:

```bash
npm run build
```

Build dosyaları `dist/` klasörüne oluşturulacaktır.

## Özellikler

- Modern glassmorphism tasarım
- Su damlası animasyonları
- Responsive tasarım
- Smooth scroll navigasyon
- Tailwind CSS ile stil
- Lucide Icons
